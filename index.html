Tag Name Description
<head> Head The Head element contains general information about an HTML page that isnâ€™t displayed on the page itself.
<link> Link The link element is used to connect the document to an outside resource, most commonly stylesheets, via the head element.
<body> Body The body element represents the content of an HTML document. Content inside body tags are rendered on the web browsers.
<a> Anchor The a anchor element is used to create hyperlinks in an HTML document.
<p> Paragraph The p paragraph element contains and displays a block of text.
<ul> Unordered List Creates an unordered list and uses <li> nested inside the <ul> for each item listed.
<ol> Ordered List Creates an ordered list and uses <li> nested inside the <ol> for each item listed.
<video> Video The video element embeds a media player for video playback. The src attribute will contain the URL to the video.
<img> Image HTML image img elements embed images in documents. The src attribute contains the image URL and is mandatory.
