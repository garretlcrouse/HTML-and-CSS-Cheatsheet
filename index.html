HTML and CSS Cheatsheet

Tag Name Description
<head> Head The <Head> element contains general information about an HTML page that isnâ€™t displayed on the page itself.
<link> Link The <link> element is used to connect the document to an outside resource, most commonly stylesheets, via the <head> element.
<body> Body The <body> element represents the content of an HTML document. Content inside <body> tags are rendered on the web browsers.
<a> Anchor The <a> element is used to create hyperlinks in an HTML document.
<p> Paragraph The <p> element contains and displays a block of text.
<ul> Unordered List Creates an unordered list and uses <li> nested inside the <ul> for each item listed.
<ol> Ordered List Creates an ordered list and uses <li> nested inside the <ol> for each item listed.
<video> Video The <video> element embeds a media player for video playback. The src attribute will contain the URL to the video.
<img> Image A <img> element embed images in documents. The src attribute contains the image URL and is mandatory.
<figure> Figure The <figure> element is used to encapsulate media such as an image, diagram. or code snippet.
<figcaption> Figure Caption The <figcaption> element is used to describe the media encapsulated within the <figure> element.
<section> Section A <section> defines elements in a document, such as chapters, headings, or any other area of the document with the same theme.
<article> Article A <article> holds content that makes sense on its own such as articles, blogs, and comments.
<nav> Navigation A <nav> is used to define a block of navigation links such as menus and tables of contents. It is important to note that <nav> can be used inside of the <header> element but can also be used on its own.
